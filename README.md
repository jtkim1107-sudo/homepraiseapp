# 우리집 칭찬가게 🍪

가족용 칭찬 쿠키 앱. 부모가 아이에게 **약속**을 보내고, 아이가 지키면 **쿠키**를 지급하고, 아이는 모은 쿠키로 **쿠키마켓**에서 원하는 보상과 바꿉니다.

## 핵심 루프

```
부모: 약속 보내기 → 아이: 수행 후 "했어요!" → 엄마: 확인 + 쿠키 지급
→ 아이: 쿠키통에 쿠키 적립 → 아빠 쿠키마켓에서 교환
```

## 역할

| 사용자 | 역할 |
|---|---|
| 엄마 | 약속 확인(승인) + 쿠키 지급 전담 |
| 아빠 | 쿠키마켓 운영 전담 (보상 추가·가격·삭제) |
| 아이들 | 약속 지키고 쿠키 모아 보상 교환 |

약속 만들기는 부모 둘 다 가능. 가족 이름은 앱 안 ⚙️ 설정에서 변경할 수 있어요.

## 게시판 📌

가족 모두가 쓰는 소통 공간이에요. 하고 싶은 말이나 고마운 마음을 글로 남기고, 💛 하트로 반응할 수 있어요. 글은 본인(부모님은 모두) 삭제 가능.

## 가족방과 보안 🔑

- 처음 접속하면 **새 가족방 만들기** 또는 **초대 코드로 들어가기**를 선택해요.
- 가족방을 만들면 초대 코드(예: `ABCD-2345`)가 생기고, 설정 ⚙️에서 확인할 수 있어요.
- 다른 기기에서 그 코드를 입력하면 같은 가족방에 연결돼 실시간 동기화돼요.
- 초대 코드 = 가족방 열쇠. **가족 외에는 알려주지 마세요.**

### Firebase 보안 규칙 적용 (최초 1회, 중요!)

`firebase-rules.json`의 내용을 Firebase 콘솔에 붙여넣어야 다른 사람이 가족방 목록을 훑어보는 것을 막을 수 있어요:

1. [Firebase 콘솔](https://console.firebase.google.com) → `homepraiseapp` 프로젝트
2. 왼쪽 메뉴 **빌드 → Realtime Database** → **규칙** 탭
3. 기존 내용을 지우고 `firebase-rules.json` 파일 내용을 붙여넣기
4. **게시** 버튼 클릭

적용 후에는: 방 목록 조회 불가(코드를 정확히 알아야 접근), 상태 데이터 형식·크기 검증.

## 실행

정적 웹앱입니다. `index.html`을 브라우저로 열거나, 로컬 서버로:

```powershell
powershell -ExecutionPolicy Bypass -File serve.ps1   # http://localhost:5173
```

- 데이터는 Firebase Realtime Database로 **가족 기기 간 실시간 동기화**됩니다.
- 오프라인이면 LocalStorage로 동작하고, 연결되면 다시 동기화됩니다.
- 부모님 화면은 비밀번호(숫자 4자리, 기본 0000)로 잠겨 있습니다.
- 매일 약속은 자정(KST) 기준으로 다시 생깁니다.
- 반응형: 폰(<600px) / 태블릿(≥600px) 레이아웃.

## 기술

HTML + CSS + Vanilla JS (프레임워크 없음) + Firebase Realtime Database. 폰트: Jua, Pretendard.
