# 우리집 칭찬가게 🍪

가족용 칭찬 쿠키 앱. 부모가 아이에게 **약속**을 보내고, 아이가 지키면 **쿠키**를 지급하고, 아이는 모은 쿠키로 **달란트시장**에서 원하는 보상과 바꿉니다.

## 핵심 루프

```
부모: 약속 보내기 → 아이: 수행 후 "했어요!" → 엄마: 확인 + 쿠키 지급
→ 아이: 쿠키통에 쿠키 적립 → 아빠 달란트시장에서 교환
```

## 역할

| 사용자 | 역할 |
|---|---|
| 엄마 | 약속 확인(승인) + 쿠키 지급 전담 |
| 아빠 | 달란트시장 운영 전담 (보상 추가·가격·삭제) |
| 아이들 | 약속 지키고 쿠키 모아 보상 교환 |

약속 만들기는 부모 둘 다 가능. 가족 이름은 앱 안 ⚙️ 설정에서 변경할 수 있어요.

## 게시판 📌

가족 모두가 쓰는 소통 공간이에요. 하고 싶은 말이나 고마운 마음을 글로 남기고, 💛 하트로 반응할 수 있어요. 글은 본인(부모님은 모두) 삭제 가능.

## 실행

정적 웹앱입니다. `index.html`을 브라우저로 열거나, 로컬 서버로:

```powershell
powershell -ExecutionPolicy Bypass -File serve.ps1   # http://localhost:5173
```

- 데이터는 Firebase Realtime Database로 **가족 기기 간 실시간 동기화**됩니다.
- 오프라인이면 LocalStorage로 동작하고, 연결되면 다시 동기화됩니다.
- 부모님 화면은 비밀번호(숫자 4자리, 기본 0000)로 잠겨 있습니다.
- 매일 약속은 자정(KST) 기준으로 다시 생깁니다.
- 반응형: 폰(<600px) / 태블릿(≥600px) 레이아웃.

## 기술

HTML + CSS + Vanilla JS (프레임워크 없음) + Firebase Realtime Database. 폰트: Jua, Pretendard.
